高中数学-复数

复数

虚数单位

符号：\(\rm i\)
定义：\(\text{i}^2 = -1\)
\(x^2 = -1\) 的根为 \(x = \pm \text{i}\)

复数定义及相关概念

\(z = a + bi\ (a, b \in \mathbb{R})\)
\(a = Re(z)\) 称为实部
\(b = Im(z)\) 称为虚部

\(b = 0 \Rightarrow\) 实数
\(b \neq 0 \Rightarrow\) 虚数
\(\left\{\begin{gather*}a = 0 \\b \neq 0\end{gather*}\right. \Rightarrow\) 纯虚数

复数集：\(\mathbb{C}\)

复数运算法则

\(z = a + b\text{i}, z_1 = a + b\text{i}, z_2 = c + d\text{i}\)

加减：\(z_1 \pm z_2 = (a\pm c) + (b\pm d)i\)

乘法： \[ \begin{align*} z_1 \times z_2 & = ac + bc\text{i} + ad\text{i} + bd\text{i}^2 \\ & = (ac - bd) + (bc + ad)\text{i} \end{align*} \]

共轭复数：\(\bar{z} = a - b\text{i}\)

除法： \[ \begin{align*} \frac{z_1}{z_2} = \frac{z_1\overline{z_2}}{z_2\overline{z_2}} & = \frac{(a+b\text{i})(c-d\text{i})}{(c+d\text{i})(c-d\text{i})} (分母实数化) \\ & = \frac{(ac + bd) + (bc - ad)\text{i}}{c^2 + d^2} \end{align*} \]

复数的模：
\(|z| = \sqrt{a^2 + b^2}\)
则有 \(z \times \bar{z} = a^2 + b^2 = |z|^2\)

复数的三角形式

\(z = a + b\text{i}\)，复平面上一点 \(Z(a, b)\)

模长：\(r = |z| = OZ = \sqrt{a^2 + b^2}\)
辐角：\(\theta\) 是以 \(x\) 轴正半轴为始边，射线 \(OZ\) 为终边的角
（辐角有无穷多个，在 \([0, 2\pi)\) 内的辐角称为辐角主值，记为 \(\arg z\)）

可知 \(z = r(\cos\theta + \text{i}\sin\theta)\)

复数的不同形式

\[ \begin{array}{c|c|c|c} 代数 & 几何 & 向量 & 三角 & 指数(由欧拉公式) \\ \hline a+b\text{i} & Z(a, b) & \overrightarrow{OZ} & r(\cos\theta + \text{i}\sin\theta) & r\!\cdot \!e^{\text{i}\theta} \end{array} \]

棣莫弗公式

\(z_1 = r_1(\cos\theta_1 + \text{i}\sin\theta_1), z_2 = r_2(\cos\theta_2 + \text{i}\sin\theta_2)\)

\[ \begin{align*} z_1 \times z_2 & = r_1r_2(\cos\theta_1 + \text{i}\sin\theta_1)(\cos\theta_2 + \text{i}\sin\theta_2) \\ & = r_1r_2[(\cos\theta_1\cos\theta_2 - \sin\theta_1\sin\theta_2) + \text{i}(\sin\theta_1\cos\theta_2 + \cos\theta_1\sin\theta_2)] \\ & = r_1r_2[cos(\theta_1 + \theta_2) + \text{i}\sin(\theta_1 + \theta_2)] \end{align*} \]

  1. 两复数积（商）的模等于模的积（商）
  2. 两复数积的辐角等于辐角的和

乘方形式

\((\cos\theta + \text{i}\sin\theta)^n = \cos n\theta + \text{i}\sin n\theta\)

几何意义

\(z_1 \times (\cos\theta + \text{i}\sin\theta) = z_2 \iff z_1\) 旋转 \(\theta\) 得到 \(z_2\)

\((x, y)\) 逆时针旋转 \(\theta\) 得到 \((x', y')\)，其中 \(\left\{\begin{gather*}x' = x\cos\theta - y\sin\theta \\ y' = x\sin\theta + y\cos\theta\end{gather*}\right.\)

矩阵形式：

\[ \begin{pmatrix} x \\ y \end{pmatrix} \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix} = \begin{pmatrix} x' \\ y' \end{pmatrix} \]

一些技巧

  1. 代数基本定理：\(n\) 次方程有 \(n\) 个根（包括重根）

  2. 实系数方程的虚根成对出现，即若 \(z\) 满足一实系数方程，\(\bar{z}\) 也满足该方程

  3. 求和形式，通过旋转，转到同一方向后，便可使用 \(|z_1 + z_2| = |z_1|+|z_2|\) 进行变形

  4. 多项式相关题目，通过适当的旋转，将某一项转到 \(x\) 轴正半轴上

  5. 多项式相关题目，通过旋转和换元，将常数项和其他某一项系数变为非负实数

单位复数

\(z = \cos\theta + \text{i}\sin\theta\)，即 \(|z| = 1\)
有 \(z\cdot\bar{z} = |z|^2 = 1\)

单位根

\(z^n = 1 \ (n \in \mathbb{N} 且 n \neq 1)\) 有 \(n\) 个根，且具有后续提到的性质
\(n\) 个根为： \(\varepsilon_k = \cos\frac{2k\pi}{n} + \text{i}\sin\frac{2k\pi}{n}, k \in \{0, 1, 2, \cdots, n-1\}\)

证明：

\(z^n = 1 \Rightarrow |z| = 1\)
设 \(z = \cos\theta + \text{i}\sin\theta\)
\(z^n = \cos n\theta + \text{i}\sin n\theta = \cos0 + \text{i}\sin0\)
\(\Rightarrow n\theta = 2k\pi \Rightarrow \theta = \frac{2k\pi}{n}, k \in \{0, 1, 2, \cdots, n-1\}\)

性质

  1. \(\forall k \in \{0, 1, 2, \cdots, n-1\}, \varepsilon_k = \varepsilon_1^k\)

  2. \(\sum_{k=0}^{n-1}\varepsilon_k = 0\)

证明：

\(\varepsilon_1^n - 1 = 0 \Rightarrow (\varepsilon_1-1)(\varepsilon_1^{n-1} + \varepsilon_1^{n-2} + \cdots + \varepsilon_1 + 1) = 0\)
\(\because \varepsilon_1 - 1 \neq 0\)
\(\therefore \sum_{k=0}^{n-1}\varepsilon_1^k = 0\)，即 \(\sum_{k=0}^{n-1}\varepsilon_k = 0\)

  1. \(\sum_{k=0}^{n-1}\varepsilon_k^r = 0 \ (n \nmid r)\)

证明：

与性质2同理

\((\varepsilon_1^r)^n - 1 = 0 \Rightarrow (\varepsilon_1^r-1)(\varepsilon_1^{r(n-1)} + \varepsilon_1^{r(n-2)} + \cdots + \varepsilon_1^r + 1) = 0\)
\(\because \varepsilon_1^r - 1 \neq 0\)
\(\therefore \sum_{k=0}^{n-1}\varepsilon_1^{rk} = 0\)，即 \(\sum_{k=0}^{n-1}\varepsilon_k^r = 0\)

  1. \(n\) 个单位根在复平面上构成正 \(n\) 边形

欧拉公式

\(e^{\text{i}\theta} = \cos\theta + \text{i}\sin\theta\)
