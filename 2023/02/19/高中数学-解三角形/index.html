高中数学-解三角形

解三角形

记 \(\triangle ABC\) 中 \(A, B, C\) 的对边分别为 \(a, b, c\)

正弦定理

\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} ( = 2R)(R 为 \triangle ABC 外接圆半径) \]

证明

作 \(AH \perp BC\) ，垂足为 \(H\)，记 \(h = AH\)
对 \(B\) 进行讨论

锐角

在 \(\text{Rt}\triangle AHB\) 中，有 \(h = c\sin B\) 在 \(\text{Rt}\triangle AHC\) 中，有 \(h = b\sin C\) 结合两式有 \(\frac{b}{\sin B} = \frac{c}{\sin C}\)

直角

此时 \(B, H\) 重合，\(h = c\) \(\because B = \frac\pi2\) \(\therefore h = b\sin C, \sin B = 1\) \(\therefore b\sin C = c = c\sin B \Rightarrow \frac{b}{\sin B} = \frac{c}{\sin C}\)

钝角

在 \(\text{Rt}\triangle AHB\) 中，有 \(h = c\sin (\pi - B) = c\sin B\) 在 \(\text{Rt}\triangle AHC\) 中，有 \(h = b\sin C\) 结合两式有 \(\frac{b}{\sin B} = \frac{c}{\sin C}\)

同理可得 \(\frac{a}{\sin A} = \frac{b}{\sin B}\)

余弦定理

\[ \begin{align*} a^2 = b^2 + c^2 - 2bc\cos A \\ b^2 = a^2 + c^2 - 2ac\cos B \\ c^2 = a^2 + c^2 - 2ab\cos C \end{align*} \]

证明

向量法

\[ \begin{align*} \overrightarrow{BC} & = \overrightarrow{BA} + \overrightarrow{AC} \\ \overrightarrow{BC}^2 & = (\overrightarrow{BA} + \overrightarrow{AC})^2 \\ |\overrightarrow{BC}|^2 & = |\overrightarrow{BA}|^2 + |\overrightarrow{AC}|^2 + 2|\overrightarrow{BA}||\overrightarrow{AC}|\cos<\overrightarrow{BA},\overrightarrow{AC}> \\ a^2 & = b^2 + c^2 - 2bc\cos A \end{align*} \]

同理可得其余两式

变形

\[ \begin{align*} \cos A = \frac{b^2 + c^2 - a^2}{2bc} \\ \cos B = \frac{a^2 + c^2 - b^2}{2ac} \\ \cos C = \frac{a^2 + b^2 - c^2}{2ab} \end{align*} \]

用于解三角形

简单形式： 给出 \(A, B, C, a, b, c\) 中三个量的值，求其他若干个

枚举后可知共有 \(6\) 种可能，分别为 \(\text{AAA, AAS, ASA, SSA, SAS, SSS}\) 其中 \(\text{AAA}\) 有无穷多组解

\(\text{AAS}\)

不妨设已知 \(A, B, a\)
由 \(A+B+C = \pi\) 得 \(C\)，再用正弦定理得 \(b, c\)

\(\text{ASA}\)

不妨设已知 \(A, B, c\)
由 \(A+B+C = \pi\) 得 \(C\)，再用正弦定理得 \(a, b\)

\(\text{SSA}\)

不妨设已知 \(a, b, A\)

解的数量：

对 \(A\) 进行讨论

锐角
  1. \(a < b\sin A\) 无解
  2. \(a = b\sin A\) 有唯一解，此时 \(B = \frac\pi2\)
  3. \(b\sin A < a < b\) 有两解
  4. \(a \geq b\) 有唯一解
直角或钝角
  1. \(a \leq b\) 无解
  2. \(a > b\) 有唯一解

求解：

正弦求 \(B\)，再求 \(C, c\)
也可余弦求 \(c\)，再求 \(B, C\)

\(\text{SAS}\)

不妨设已知 \(A, b, c\) 由余弦定理得 \(a\), 之后正弦 \(/\) 余弦均可

\(\text{SSS}\)

由余弦定理得任意一角 \(\cos\)，之后正弦 \(/\) 余弦均可
